# Retail Sales Intelligence Dashboard (BI + ML)

## Project Overview
This project demonstrates an end-to-end Business Intelligence pipeline using retail sales data, integrating ETL, database modeling, machine learning, and interactive dashboards.

## Tech Stack
- Python (Pandas, Scikit-learn)
- PostgreSQL
- Power BI
- Machine Learning (RFM + K-Means, Time Series Forecasting using Prophet)

## Project Structure
├── etl/
├── ml/
├── utils/
├── db/
├── requirements.txt
└── README.md

## Key Features
- ETL pipeline for data ingestion and cleaning
- Star schema design with fact and dimension tables
- Customer segmentation using RFM + K-Means
- Sales forecasting with confidence intervals
- Interactive Power BI dashboards

## Dashboards
- Executive Sales Overview
- Customer Segmentation & Value Analysis
- Sales Forecast & Future Outlook
